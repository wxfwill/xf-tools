xf-tools工具集合

  • 通用类方法

    获取url参数: urlParse(str:string)

    str: url地址

  • 日期类方法

    格式化时间戳: formatDate(date: Date, format: 'yyyy-MM-dd')

    date: new Date日期类型, format: 格式化的形式，支持多种组合 yyyy、MM、dd、hh、ss、mm

  • 下载类方法

    文件流下载: downFileBlob(file:blob, name:string, type?: string)

    file: 文件流, name: 文件名字， type: 文件类型，默认pdf

    url下载: downFileUrl(url: string, name:string)

    url: 文件的url, name: 文件名字

  • 打印类方法

    html打印dom htmlPrint(id:string)

    id: dom的id

    根据url打印 urlPrint(url: string, isbase64?: boolean)

    url: 文件的url地址， isbase64：是否采用base64打印
